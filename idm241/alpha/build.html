IDM 241: Alpha Build - Zillow.com

Interaction: Favorite Button

Trigger

Favorite Button - Default

  • hover ON
  • hover OFF
  • mouse DOWN
  • mouse UP (click ON)

Favorite Button - Active

  • hover ON
  • hover OFF
  • mouse DOWN
  • mouse UP (click OFF)

Rules

Favorite Button - Default

  • hover ON
    • Cursor changes to pointer
    • Filled, black opacity in icon changes to 50%
  • hover OFF
    • Cursor changes back to default
    • Filled, black opacity in icon changes back to 25%
  • mouse DOWN
    • filled, black opacity in icon changes to 50%
  • mouse UP (click ON)
    • Icon changes to filled, white heart
    • Database updates

Favorite Button - Active

  • hover ON
    • Cursor changes to pointer
    • Blue, rounded square appears around icon
  • hover OFF
    • Cursor changes back to default
    • Blue, rounded square disappears from icon
  • mouse DOWN
    • None
  • mouse UP (click OFF)
    • Icon changes back to white, outlined heart
    • Filled, black opacity in icon changes to 50%
    • Database updates
