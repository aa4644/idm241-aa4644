IDM 241: Beta Description - Zillow.com

Interaction: Favorite Button, Property Modal Card, "Request Tour" Button, "X" Close Modal Button

Triggers

Favorite Button - Default

  • hover ON
  • hover OFF
  • mouse DOWN
  • mouse UP (click ON)

Favorite Button - Active

  • hover ON
  • hover OFF
  • mouse DOWN
  • mouse UP (click OFF)

Property Modal Card - Default

  • hover ON
  • hover OFF
  • mouse DOWN
  • mouse UP (click ON)

"Request Tour" - Default

  • hover ON
  • hover OFF
  • mouse DOWN
  • mouse UP (click ON)

"X" Close Modal Button - Default

  • hover ON
  • hover OFF
  • mouse UP (click ON)

Rules

Favorite Button - Default

  • hover ON
    • Cursor changes to pointer
    • Outline changes from white to red
  • hover OFF
    • Cursor changes back to default
    • Outline changes back to white
  • mouse DOWN
    • Red glow appears around icon
  • mouse UP (click ON)
    • Icon changes to filled, heart heart
    • Database updates

Favorite Button - Active

  • hover ON
    • Cursor changes to pointer
    • Filled icon changes to darker red
  • hover OFF
    • Cursor changes back to default
    • Filled icon changes back to original red
  • mouse DOWN
    • Red glow appears around icon
  • mouse UP (click OFF)
    • Icon changes back to white, outlined heart
    • Database updates

Property Modal Card - Default

  • hover ON
    • Cursor changes to pointer
    • Card enlarges 25%
    • Black shadow at 25% opacity appears around card
    • "Request Tour" button slides in from the right side of the card
  • hover OFF
    • Cursor changes back to default
    • Card returns to original size
    • Black shadow around card disappears
    • "Request Tour" button slides out to the right, hiding from view
  • mouse DOWN
    • Black shadow at 50% opacity appears around card
  • mouse UP (click ON)
    • User navigates to the property details page

"Request Tour" Button - Default

  • hover ON
    • Cursor changes to pointer
    • Button background fills with blue at 25% opacity
  • hover OFF
    • Cursor changes back to default
    • Button background returns to tranparent filling
  • mouse DOWN
    • Button background fills with blue at 50% opacity
  • mouse UP (click ON)
    • "Request Tour" modal opens, allowing user to select and enter their preferred tour date and time

"X" Close Modal Button - Default

  • hover ON
    • Cursor changes to pointer
  • hover OFF
    • Cursor changes back to default
  • mouse UP (click ON)
    • Pop-up modal closes
