Rules

Favorite Button - Default

  • hover ON
    • Cursor remains as pointer
    • Outline changes from white to red
  • hover OFF
    • Cursor remains as pointer
    • Outline changes back to white
  • mouse UP
    • Icon changes to filled, red heart
    • Database updates

Favorite Button - Active

  • hover ON
    • Cursor remains as pointer
    • Red glow appears around icon
  • hover OFF
    • Cursor remains as pointer
    • Red glow around icon disappears
  • mouse UP
    • Icon changes back to white, outlined heart
    • Database updates

Property Modal Card

  • hover ON
    • Cursor remains as pointer
    • Card enlarges 25%
    • Black shadow at 25% opacity appears around card
    • "Request Tour" button appears
  • hover OFF
    • Cursor remains as pointer
    • Card returns to original size
    • Black shadow opacity turns to 10% around card
    • "Request Tour" button disappears

"Request Tour" Button

  • hover ON
    • Cursor remains as pointer
    • Button background fills with blue at 25% opacity
  • hover OFF
    • Cursor remains as pointer
    • Button background returns to tranparent filling
  • mouse DOWN
    • Button background fills with blue at 50% opacity
  • mouse UP
    • "Request Tour" modal opens, allowing user to select and enter their preferred tour date and time

"X" Close Modal Button

  • hover ON
    • Cursor remains as pointer
    • Icon changes from black to blue
  • hover OFF
    • Cursor remains as pointer
    • Icon changes back to black
  • mouse UP
    • Pop-up modal closes
