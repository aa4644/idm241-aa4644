Rules

Favorite Button - Default

  • hover ON
    • Cursor remains as pointer
    • Outline changes from white to red
  • hover OFF
    • Cursor remains as pointer
    • Outline changes back to white
  • mouse DOWN
    • Red glow appears around icon
  • mouse UP
    • Icon changes to filled, red heart
    • Database updates

Favorite Button - Active

  • hover ON
    • Cursor remains as pointer
    • Red glow appears around icon
  • hover OFF
    • Cursor remains as pointer
    • Red glow around icon disappears
  • mouse DOWN
    • None
  • mouse UP
    • Icon changes back to white, outlined heart
    • Database updates

Property Modal Card

  • hover ON
    • Cursor remains as pointer
    • Card enlarges 25%
    • Black shadow at 25% opacity appears around card
    • "Request Tour" button fades in
  • hover OFF
    • Cursor remains as pointer
    • Card returns to original size
    • Black shadow opacity turns to 10% around card
    • "Request Tour" button fades out

"Request Tour" Button

  • hover ON
    • Cursor remains as pointer
    • Button background fills with blue at 25% opacity
  • hover OFF
    • Cursor remains as pointer
    • Button background returns to tranparent filling
  • mouse DOWN
    • Button background fills with blue at 50% opacity
  • mouse UP
    • "Request Tour" modal opens, allowing user to select and enter their preferred tour date and time

"X" Close Modal Button

  • hover ON
    • Cursor remains as pointer
    • Icon changes from black to blue
  • hover OFF
    • Cursor remains as pointer
    • Icon changes back to black
  • mouse UP
    • Pop-up modal closes

Date Selection Button - Default

  • hover ON
    • Cursor changes to pointer
    • Button background fills with black at 25%
    • Black shadow at 25% opacity appears around button
  • hover OFF
    • Cursor changes to back to default
    • Button background returns to transparent filling
    • Black shadow disappears
  • mouse DOWN
    • Button background fills with blue at 50% opacity
    • Blue shadow at 50% opacity appears around button
    • Text and outline changes from black to blue
  • mouse UP
    • Button background fills with blue at 25% opacity
    • Blue shadow at 25% opacity appears around button
    • Datebase updates

Date Selection Button - Active

  • hover ON
    • Cursor changes to pointer
    • Blue shadow at 50% opacity appears around button
  • hover OFF
    • Cursor changes to back to default
    • Blue shadow changes back to 25% opacity around button
  • mouse DOWN
    • button background fills with blue at 50% opacity
    • blue shadow at 50% opacity appears around button
  • mouse UP
    • Button background returns to transparent filling
    • Blue shadow disappears, text and outline changes from blue to black
    • Database updates

Image Scroll Buttons

  • hover ON
    • Cursor remains as pointer
    • Arrows and page image indicator appears
  • hover OFF
    • Cursor remains as pointer
    • Arrows and page image indicator disappears
  • mouse UP
    • image slides to the left if clicked using right arrow and slides to the right if clicked using left arrow
    • Page indicator switches to what image number page user is on
