Rules

Property Modal Card

  • hover ON
    • Cursor remains as pointer
    • Card enlarges 25%
    • Black shadow at 25% opacity appears around card
    • "Request Tour" button fades in
  • hover OFF
    • Cursor remains as pointer
    • Card returns to original size
    • Black shadow opacity turns to 10% around card
    • "Request Tour" button fades out

Favorite Button - Default

  • hover ON
    • Cursor remains as pointer
    • Outline changes from white to red
  • hover OFF
    • Cursor remains as pointer
    • Outline changes back to white
  • mouse DOWN
    • Red glow appears around icon
  • mouse UP
    • Icon changes to filled, red heart
    • Database updates

Image Scroll Buttons

  • hover ON
    • Cursor remains as pointer
    • Arrows and page image indicator appears
  • hover OFF
    • Cursor remains as pointer
    • Arrows and page image indicator disappears
  • mouse UP
    • Image slides to the left if clicked using right arrow and slides to the right if clicked using left arrow
    • Page indicator switches to what image number page user is on
