IDM 241: Alpha - x.com

Interaction: Like Button

Trigger

  • hover ON
  • hover OFF
  • click ON
  • click OFF

Rules

Like Button - Default

  • hover ON - Cursor changes to pointer
  • hover OFF - Cursor changes back to default
  • click ON
    • Icon changes to filled heart
    • Confetti animation triggers
    • Database record flag is set to TRUE
  • click OFF
    • Icon changes back to outline heart
    • Database record flag is set to FLASE

    Like Button - Active

    • hover ON - Cursor changes to pointer
    • hover OFF - Cursor changes back to default
    • click ON
      • Icon remains filled heart
      • No additional action can be taken
    • click OFF
      • Icon changes back to outline heart
      • Confetti animation triggers
      • Database record flag is set to FALSE

Feedback

Like Button - Default

  • hover ON - Cursor changes to pointer
  • hover OFF - Cursor returns to default
  • click ON
    • Confetti animation plays
    • Optional sound effect (e.g. "pop") may play to indicate the action ??
  • click OFF - No animation; icon visually updates to outline heart

Like Button - Active

  • hover ON - Cursor changes to pointer
  • hover OFF - Cursor returns to default
  • click ON - Filled heart remains; user receives visual confirmation via notifications??
  • click OFF - Confetti animation plays again, with optional sound ??

Loops & Modes

  • No Loops
