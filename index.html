IDM 241: Alpha Description - x.com

Interaction: Like Button

Trigger

  • hover ON
    • 1 second delay
  • hover OFF
  • mouse DOWN
  • mouse UP (click ON)

Rules

Like Button - Default

  • hover ON
    • Cursor changes to pointer
    • Icon outline changes from grey to red and red ring with 25% opacity radiates around icon
    • Text prompt "Like" appears after 1 second
  • hover OFF
    • Cursor changes back to default
    • Icon outline changes from red back to grey
    • Text prompt disappears
  • mouse DOWN
    • Red ring opacity increases to 75%
  • mouse UP (click ON)
    • Icon changes to filled heart and number updates
    • Confetti animation triggers

Like Button - Active

  • hover ON
    • Cursor changes to pointer
    • Red ring with 25% opacity around heart icon appears
    • Text prompt "Unlike" appears after 1 second
  • hover OFF
    • Cursor changes back to default
    • Text prompt disappears
  • mouse DOWN
    • Red ring opacity increases to 75%
  • mouse UP (click ON)
    • Icon changes back to grey heart outline
