IDM 241: Alpha - x.com

Interaction: Like Button

Trigger

  • hover ON
    • 1 second delay
  • hover OFF
  • click ON
  • click OFF

Rules

Like Button - Default

  • hover ON
    • Cursor changes to pointer
    • Icon outline changes from grey to red and radiates around icon
    • Icon and number shifts slightly upward
    • Text prompt "Like" appears after 1 second
  • hover OFF
    • Cursor changes back to default
    • Icon outline changes from red back to grey
    • Icon and number shift back to original spot
    • Text prompt disappears
  • click ON
    • Icon changes to filled heart and number updates
    • Confetti animation triggers
  • click OFF
    • Icon changes back to outline grey heart

Like Button - Active

  • hover ON
    • Cursor changes to pointer
    • Red ring with slight opacity around heart icon appears
    • Text prompt "Unlike" appears after 1 second
  • hover OFF
    • Cursor changes back to default
    • Text prompt disappears
  • click ON
    • Icon changes back to outline grey heart
